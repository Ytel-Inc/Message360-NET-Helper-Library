Welcome to the Message360 .NET Helper Library

This is home to the Official .NET Message360 REST API and InboundXML Libraries.

Example:

var random = new Random();
var newSms = Client.SMS_SendSMS("949xxxyyyy", "949xxxzzzz", "Hello world" + random.Next(500), "1", "1");
Console.WriteLine(newSms.ToString());
